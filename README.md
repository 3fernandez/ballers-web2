# ![Ballers](images/ballers-logo.png)

A basketball social network. Find basketball games near to you.

### Prerequisites
* `Git`
* `Nodejs 6.9.x LTS`
* [`Yarn`](https://yarnpkg.com/)


### Getting Started
  ```
  yarn
  yarn start
  ```
